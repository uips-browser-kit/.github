# uips-browser-kit

Collaborative browser automation toolkit for UiPath.

## Repositories

| Repository | Purpose |
|---|---|
| [library](https://github.com/uips-browser-kit/library) | Shipped UiPath browser automation package |
| [rules](https://github.com/uips-browser-kit/rules) | Workflow Analyzer policy — custom rules and configuration |
| [snippets](https://github.com/uips-browser-kit/snippets) | Reusable browser automation snippets and reference patterns |
| [testproject](https://github.com/uips-browser-kit/testproject) | UiPath consumer project for manual/regression tests |
| [testdata](https://github.com/uips-browser-kit/testdata) | Reusable fixtures, profiles, and sample pages |
| [docs](https://github.com/uips-browser-kit/docs) | Contributor and user documentation |
| [testharness-webapps](https://github.com/uips-browser-kit/testharness-webapps) | Scripts and tools for repeatable checks |
| [playground](https://github.com/uips-browser-kit/playground) | Safe experiments for users and contributors |
| [exploration](https://github.com/uips-browser-kit/exploration) | Research spikes, discarded ideas, and findings |
| [discussions](https://github.com/uips-browser-kit/discussions) | Community discussions |
